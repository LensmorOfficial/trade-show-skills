# Booth Invitation Writer

Generate personalized pre-show invitation emails that actually get replies — tailored by audience type and relationship.

## What It Does

Provide your show details and audience type. Claude writes:

- Personalized invitation emails (not generic "visit our booth" blasts)
- Subject line A/B variants
- Optional follow-up reminders
- Multi-language support for international shows

Supports different templates for cold prospects, existing customers, partners, and VIP/executive outreach.

## Usage

```
Write a booth invitation email for MEDICA 2026, booth 5C42. We're launching a new surgical robot.
```

```
I need to invite 3 types of people to our Interpack booth: existing distributors, new prospects, and press contacts.
```

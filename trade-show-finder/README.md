# Trade Show Finder

Find and compare trade shows by industry, region, and date — powered by web search for up-to-date results.

## What It Does

Give Claude your industry, target region, and timeframe. It searches the web for matching trade shows and returns:

- A structured comparison table (dates, location, exhibitor/visitor counts, focus areas)
- Top picks with specific recommendations on why each show fits your needs
- Practical context: registration deadlines, co-located events, travel tips

## Usage

```
Find packaging trade shows in Europe for Q2-Q3 2026
```

```
I sell industrial automation equipment. What are the biggest shows in Asia this year?
```

```
Compare CES, MWC, and IFA for a consumer electronics startup
```
